# Dorel - Backend Challenge

## Tools
- Ruby 3.2.2
- Rails 7.0.8
- PostgreSQL 14

## Setup
- Run the following commands:

```
cd dorel-backend-challenge
bundle install
rails db:setup
rails s
```
